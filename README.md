Advanced Calculator, with some special operations.
Supported operators:
	 Binary Operators:
     Addition      +  // 1+2=3
	   Subtract      -  // 2-1=1
	   Multiply      *  // 5*3=15
	   Divide        /  // 10/5=2
	   Power         ^  // 2^10=1024
	   Modulo        %  // 10%4=2
	   Maximum       $  // 5$7=7
	   Minimum       &  // 10&3=3
	   Average       @  // 10@20=15
   Unary Operators:
	 Left Unary Operators:
     Negative      ~  // ~3=-3
   Right Unary Operators:
	   Sum of digits #  // 1.52#=8
     Factorial     !  // 5!=120

Extra features:
	 Parenthesis, decimal point, unary minus
